复古风格效果使图像看上去就和老照片一样。

这是一张没有经过处理的图片:

![Image without effect][1]

And the same image with the effect applied:

![Image with effect][2]

内置的复古效果有以下的属性设置:

* **Amount**: Controls the intensity of the effect. Ranges from 0 to 1.

Find the post-processing effect script on [GitHub][3].
