Audio Listener

The Audio Listener Component specifies the listener's position in 3D space. All 3D audio playback will be relative to this position.

See Also​

  • Sound Component - Play audio in 3D space
  • Audio Assets - Learn more about audio assets

Scripting Interface​

You can control an Audio Listener Component's properties using a Script Component. The Audio Listener Component's scripting interface is here.
