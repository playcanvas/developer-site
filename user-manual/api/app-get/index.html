Apps - Get app

Route URL​

GET https://playcanvas.com/api/apps/:id

Description​

Gets a published App by id.

Example​

curl -H "Authorization: Bearer {accessToken}" https://playcanvas.com/api/apps/{id}

Parameters​

NameTypeDescription
idnumberThe id of the app.

Response Schema​

Status: 200
{
    "id": int,
    "project_id": int,
    "owner_id": int,
    "name": string,
    "description": string,
    "version": string,
    "release_notes": string,
    "thumbnails": {
        "s": string,
        "m": string,
        "l": string,
        "xl": string
    },
    "size": int,
    "views": int,
    "completed_at": date,
    "created_at": date,
    "modified_at": date
}

Errors​

CodeDescription
401Unauthorized
403Forbidden
404App not found
404Project not found
429Too many requests

Rate Limiting​

This route uses a normal rate limit.
