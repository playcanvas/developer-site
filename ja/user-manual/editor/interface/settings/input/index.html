入力設定

アプリケーションの入力デバイス処理を有効／無効にします。

注記

これらの設定は、現在アクティブなブランチの全ユーザーに影響します。

INPUT セクションに移動し、パネルを展開します。

利用可能な設定は次のとおりです。

設定​

設定説明
Keyboardキーボード入力を有効化します。無効化するとアプリはキーボード入力を無視します。
Mouseマウス入力を有効化します。無効化するとアプリはマウス入力を無視します。
Touchタッチ入力を有効化します。無効化するとアプリはタッチ入力を無視します。
Gamepadsゲームパッド入力を有効化します。無効化するとアプリはゲームパッド入力を無視します。

備考​

  • 使用しない入力方式を無効化すると、イベント処理のオーバーヘッドを削減できます。
