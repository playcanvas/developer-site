# `<Light/>`

import { CodeExample, Staging } from '@site/src/components/playcanvas-react/CodeExample';
import { Application } from '@playcanvas/react';
import { LightExample } from './light-example';
import LightExampleSource from '!!raw-loader!./light-example.jsx';

The `<Light/>` component allows an Entity to emit light. Any [`<Render/>`](../render) component will be lit by the light. When attached to an [`<Entity/>`](../entity), the light can be positioned and oriented using the entity's transform. Lights can be directional, omni or spot.

## Usage

Simply attach a `<Light/>` to an [`<Entity/>`](../entity) and it will emit light from the entity's position and rotation.

```tsx
<Entity>
  <Light type="directional" intensity={10} />
</Entity>
```
