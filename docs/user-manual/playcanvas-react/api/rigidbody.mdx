# `<Rigidbody/>`

The `<Rigidbody/>` component gives an [`<Entity/>`](../entity) physical properties such as mass, velocity, and friction and allows it to interact with the global physics simulation. An `<Entity/>` with a `<Rigidbody/>` will respond to gravity, forces, torques and other physics behaviors.

You can learn more about how the physics system works in PlayCanvas in the [Physics docs](/user-manual/physics/) and for React specific details, see the guide on [React Physics](/user-manual/playcanvas-react/guide/physics).

## Getting started

To use the `<Rigidbody/>` component, you need to install the `sync-ammo` dependancy you need to add the `<Application usePhysics/>` prop set in your root `<Application/>` component. This will enable the physics system and allow your entities to interact with the physics simulation.

```jsx
<Application usePhysics>
  <Entity>
    <Rigidbody type="box" />
  </Entity>
</Application>
```

:::tip
Using physics in PlayCanvas requires Ammos.js and `<Application usePhysics/>` to be enabled. Run `npm i sync-ammo` from within your app.
:::
