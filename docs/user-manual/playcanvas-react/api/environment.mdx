The `<Environment/>` component is used to configure the environment lighting and skybox for a scene. It provides a simple way to set up environmental lighting, skyboxes, and other effects that affect the entire scene.

## Overview

The Environment component combines several PlayCanvas scene and sky properties into a single, easy-to-use component:

- **Skybox**: Sets the background texture for the scene
- **Environment Atlas**: Provides Image-Based Lighting (IBL) for realistic environmental reflections
- **Sky Settings**: Controls sky dome properties, rotation, scale, and positioning
- **Scene Settings**: Manages exposure, luminance, and other global lighting parameters

Place the `<Environment/>` component within an [`<Application/>`](./application).


## Usage

You can use the `<Environment/>` component to set the environment lighting and skybox for a scene.

:::tip

Only one `<Environment/>` component can be used per [`<Application/>`](./application). If you use multiple `<Environment/>` components, only the first one will be applied. 

:::

### Loading a HDR

You use the `<Environment/>` component with a HDR

```jsx copy
const EnvironmentExample = () => {
  const { asset: envAtlas } = useEnvAtlas('/env_atlas.png')

  return (
    <Application>
      <Environment 
        envAtlas={envAtlas}
        exposure={1.2}
        showSkybox={false}
      />
      {/* Your scene entities here */}
    </Application>
  )
}
```
