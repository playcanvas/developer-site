Each creator on SuperSplat gets a public profile at `https://superspl.at/user?id=<username>`. The profile shows their avatar, display name, bio, any social links they've added, and a grid of every **Public** splat they've published.

Profiles are linked from the **author chip** that appears on every [scene page](/user-manual/supersplat/scene-page) and on every card in the [Explore](/user-manual/supersplat/explore) grid — clicking the author's name navigates to their profile.

Browsing a profile is anonymous; no PlayCanvas account is required.

![A creator's SuperSplat profile page — avatar, display name, social links, and a grid of their public splats](pathname:///img/user-manual/supersplat/user-profile.webp)

## What's shown

| Section | What it contains |
|---------|------------------|
| **Avatar and display name** | The creator's PlayCanvas account avatar and chosen display name. |
| **Bio** | A short profile description the creator has written about themselves. |
| **Social links** | Links to any external profiles the creator has added (X, LinkedIn, personal site, etc.). |
| **Splat grid** | Every **Public** splat the creator has published, using the same card UI as [Explore](/user-manual/supersplat/explore). Click any card to open its [scene page](/user-manual/supersplat/scene-page). |

Unlisted splats are not shown on the public profile — only the creator can see those, via the [Manage page](/user-manual/supersplat/manage).
