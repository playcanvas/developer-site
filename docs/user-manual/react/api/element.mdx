# `<Element/>`

import { CodeExample } from '@site/src/components/playcanvas-react/CodeExample';
import TextExample from '@site/src/components/playcanvas-react/examples/TextExample';
import TextExampleSource from '!!raw-loader!@site/src/components/playcanvas-react/examples/TextExample.jsx';

The `<Element/>` component draws 2D UI content — text, an image, or an invisible group — on an [`<Entity/>`](../entity). Elements live under a [`<Screen/>`](../screen), which positions them in screen or world space. Set the `type` prop to `"text"`, `"image"` or `"group"`, and use `anchor` and `pivot` to control how the element is positioned and resized within its parent.

## Usage

Add an `<Element/>` to an [`<Entity/>`](../entity) that descends from a [`<Screen/>`](../screen). For text, set `type="text"` and supply a font via `fontAsset` (load one with [`useFont`](../hooks/use-asset#usefont)).

```tsx
<Entity name="label">
  <Element
    type="text"
    fontAsset={font}
    text="Hello, World!"
    fontSize={84}
    anchor={[0.5, 0.5, 0.5, 0.5]}
    pivot={[0.5, 0.5]}
  />
</Entity>
```

The example below renders screen-space text. The font is an MSDF asset created with [font-tools](/user-manual/user-interface/fonts) and loaded with [`useFont`](../hooks/use-asset#usefont).
