---
title: Group Elements
description: Use group elements as empty containers to position and parent other UI elements in the hierarchy.
---

:::ai

- **[Editor MCP Server](/user-manual/ai/editor-mcp-server/):** Create and configure the UI entities and Components used by “Group Elements”, then launch and capture the interface.

:::

A group Element is an empty Element component that can be used as a parent for other elements. You can set the size and position of this Element and any child Elements can be positioned relative to the group.
