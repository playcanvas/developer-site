AI コーディングアシスタントは、プロジェクトファイルまたは Editor を直接操作して PlayCanvas を扱えます。タスクに適した連携方法を選択してください。

## プロジェクトファイルを操作する

AI コーディングアシスタントでスクリプト、シェーダー、その他のテキストアセットを編集する場合は、[VS Code Extension](./vscode-extension.md) を使用します。Pull/Push モードでは、アシスタントによる変更を確認して PlayCanvas プロジェクトへ Push するまでローカルに保持できます。

このワークフローは、次の用途に適しています。

- 複数のプロジェクトファイルをまとめて編集する
- プロジェクト全体の検索、型チェック、ソース管理を利用する
- コラボレーターに反映する前に変更を確認する

## Editor を自動操作する

AI アシスタントで Editor のエンティティ、アセット、シーン、ビューポート、実行中のアプリケーション、またはバージョン管理を操作する場合は、[Editor MCP Server](./editor-mcp-server.md) を使用します。

このワークフローは、次の用途に適しています。

- シーンを構築、変更する
- ビューポートと実行中のアプリケーションを確認する
- スクリーンショット、ランタイム状態、入力、コンソールログを使って変更をテストする
- アシスタントがプロジェクトを変更する前にチェックポイントを作成する
