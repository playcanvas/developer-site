**オーガニゼーション**は、チームのための共有アカウントです。プロジェクトが個人のパーソナルアカウントの下に置かれるのではなく、オーガニゼーションに所属し、オーナーと任意の数の管理者がプロジェクト、メンバー、権限をまとめて管理します。オーガニゼーションは**フリープラン**でも利用でき、必要に応じて有料プランにアップグレードできます。

## オーガニゼーションとは？

オーガニゼーションは、個人のログインではなく、チームが所有するワークスペースです。パーソナルアカウントとは次の点で異なります。

- **特定の個人に紐付きません。** オーガニゼーションとして直接ログインすることはありません。代わりに、オーナーまたは管理者として、自分自身のユーザーアカウントを通じて管理します。
- **複数の管理者を持てます。** オーナーは任意の数の管理者を指定でき、管理者はプロジェクトの作成・削除やメンバーの管理を行えます。オーガニゼーション自体を削除できるのはオーナーのみです。
- **プロジェクトはチームに帰属します。** プロジェクトは個人ではなくオーガニゼーションが所有するため、特定のメンバーのアカウントに左右されません。
- **所有権を移譲できます。** オーナーがチームを離れる場合でも、プロジェクトに影響を与えることなく、別のユーザーに所有権を引き継げます。

## オーガニゼーションとパーソナルアカウントの比較

違いは、どのプランを利用しているかではなく、**誰がアカウントを所有・管理するか**にあります。オーガニゼーションは、パーソナルアカウントと同じようにフリープランでも利用できます。

| | パーソナルアカウント | オーガニゼーション |
|---|---|---|
| 所有者 | 個人（あなた） | チームまたは企業 |
| 管理者 | 自分のみ | オーナーと任意の数の管理者 |
| プロジェクトとメンバー | プロジェクトごとに自分で共同作業者を管理 | 管理者がプロジェクトとメンバーを一元管理 |
| 適した対象 | 個人 | あらゆる規模のチーム |

## プラン

オーガニゼーションは**フリープラン**で利用でき、上記の共有アカウントとチーム管理の機能に加えて、無制限のパブリックプロジェクトを利用できます。

チームが**プライベートプロジェクト**、より大きなストレージ（50 GB）、1枚の請求書にまとめられたシート単位の請求を必要とする場合は、有料の**オーガニゼーションプラン**にアップグレードします。シートは1つあたり月額50ドルで、いつでも追加・削除でき、次回の請求日に日割りで精算されます。請求に関するFAQの[計算例](/user-manual/account-management/billing/#how-is-billing-for-organization-accounts-calculated)と、最新の詳細については[料金ページ](https://playcanvas.com/plans)をご覧ください。

## オーガニゼーションを使用するタイミング

オーガニゼーションは、次のような場合に適しています。

- プロジェクトを個人のパーソナルアカウントではなく、チームや企業に帰属させたい場合。
- プロジェクト、メンバー、権限を管理する管理者が複数人必要な場合。
- 他のメンバーと共同で作業し、共有のワークスペースが欲しい場合（フリープランでも可能です）。
