**オーガニゼーション**は、企業やチームのために設計された共有アカウントの種類です。プロジェクトが個人のパーソナルアカウントの下に置かれるのではなく、オーガニゼーションに所属し、オーナーと任意の数の管理者がメンバー、権限、請求を一元的に管理します。

## オーガニゼーションとは？

オーガニゼーションは、個人のログインではなく、企業が所有するワークスペースです。パーソナルアカウントとは次の点で異なります。

- **特定の個人に紐付きません。** オーガニゼーションとして直接ログインすることはありません。代わりに、オーナーまたは管理者として、自分自身のユーザーアカウントを通じて管理します。
- **複数の管理者を持てます。** オーナーは任意の数の管理者を指定でき、管理者はプロジェクトの作成・削除やメンバーの管理を行えます。オーガニゼーション自体を削除できるのはオーナーのみです。
- **所有権を移譲できます。** オーナーがチームを離れる場合でも、プロジェクトに影響を与えることなく、別のユーザーに所有権を引き継げます。
- **請求が一本化されます。** オーガニゼーションは**シート**単位で課金され、1枚の請求書にまとめられるため、ビジネスにとって便利です。

## オーガニゼーションとパーソナルアカウントの比較

オーガニゼーションがパーソナルアカウントと異なるのは、利用できるエディターやエンジンの基本機能ではなく、**所有・管理・請求の方法**です。共有の所有権、複数の管理者、1枚の請求書にまとめられたシート単位の請求、そしてより大きなストレージが加わるため、企業や大規模なチームに適しています。

プライベートプロジェクトを作成するには、パーソナルまたはオーガニゼーションの有料プランが必要です。フリープランでは、すべてのプロジェクトがパブリックになります。

| | パーソナルアカウント | オーガニゼーション |
|---|---|---|
| アカウントモデル | 個人のログインに紐付く | 企業が所有する共有アカウント |
| 管理者 | 自分のみ | オーナーと任意の数の管理者 |
| メンバー権限 | プロジェクトごとに自分で共同作業者を管理 | 管理者がプロジェクトごとに一元管理 |
| 請求 | 個人ユーザーのサブスクリプション | シート単位、1枚の請求書にまとめて |
| ストレージ（有料プラン） | 10 GB | 50 GB |
| 適した対象 | 個人 | 企業や大規模なチーム |

## 料金

オーガニゼーションプランは**1シートあたり月額50ドル**で、50 GBのストレージが含まれます。シートはいつでも追加・削除でき、変更は次回の請求日に日割りで精算されます。計算例については[請求に関するFAQ](/user-manual/account-management/billing/)をご覧ください。最新の詳細は[料金ページ](https://playcanvas.com/plans)をご確認ください。

:::note

年間売上が10万ドルを超えるビジネスは、オーガニゼーションを選択してください。個人や小規模なチームはパーソナルプランを利用できます。

:::

## オーガニゼーションを使用するタイミング

オーガニゼーションは、次のような場合に適しています。

- 特定の従業員に依存せず、ビジネスとしてプロジェクトを所有したい場合。
- メンバーや権限を管理する管理者が複数人必要な場合。
- 1枚の請求書にまとめられたシート単位の請求を希望する場合。
