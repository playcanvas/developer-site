[8th Wall](https://www.8thwall.com/)は、Webを通じてインタラクティブな拡張現実体験を作成するためのツールを提供するプラットフォームです。3.5B以上のユーザーに到達しており、ワールドトラッキング、イメージトラッキング、フェイストラッキングをサポートしています。2018年には、PlayCanvasとの完全な統合が追加されました。

## 8th Wallの始め方

8th Wallのドキュメンテーションには、[PlayCanvasの始め方](https://www.8thwall.com/docs/legacy/api/playcanvas/getting-started/)のページが用意されており、[スタータープロジェクト](https://playcanvas.com/user/the8thwall)も多数提供されています。
