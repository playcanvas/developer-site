Audio Listenerコンポーネントは、リスナーの3D空間での位置を指定します。 すべての 3D 音声再生はこの位置を基準に相対的に再生されます。


Audio Listenerコンポーネントは、コンポーネントパネルの右上にあるトグルを使用して有効または無効にできます。有効にすると、このコンポーネントは3D音響効果におけるリスナー位置として機能します。
