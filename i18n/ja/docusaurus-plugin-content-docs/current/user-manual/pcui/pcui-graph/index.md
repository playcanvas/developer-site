![PCUI Graph バナー](/img/user-manual/pcui/pcui-graph-banner.png)

PCUI Graph は、ブラウザでノードベースのグラフを作成および表示できるアプリケーションの構築を支援するために設計されたグラフ可視化フレームワークです。これは PCUI ライブラリの拡張機能として構築されましたが、現在は独自のレポジトリで管理されています。
