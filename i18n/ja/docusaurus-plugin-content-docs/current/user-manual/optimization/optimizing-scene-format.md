---
title: シーンフォーマットの最適化
description: Scene の JSON を圧縮して gzip 後のダウンロードを小さくし、ランタイムの挙動は変えない Publish オプションです。
---

:::ai

- **[Editor MCP Server](/user-manual/ai/editor-mcp-server/):** Scene Format の最適化を有効にして Publish または Download Build を作成し、完了を待って、生成された Build と Artifact を確認します。

:::

このパブリッシュオプションは、サーバー上でgzip圧縮された後、シーンファイルのサイズを約30-50%まで縮小することができます。

有効にするには、Publishの画面で 'Optimize Scene Format' にチェックを入れてから公開してください。
