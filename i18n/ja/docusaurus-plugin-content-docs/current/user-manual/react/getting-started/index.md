PlayCanvas React へようこそ！ このセクションでは、最初の 3D React アプリケーションのセットアップ方法と、インタラクティブな 3D 体験を構築するために必要な基本概念をガイドします。

## 学べること

このはじめにガイドでは、以下の方法を学びます:

- プロジェクトに PlayCanvas React を[インストール・セットアップ](./installation)する
- カメラ、Entity、3D モデルを使って[最初の 3D シーンを作成](../building-a-scene)する
- アプリケーションで効率的に [3D Asset をロード・管理](../guide/loading-assets)する

## 前提条件

始める前に、以下を確認してください:

- **Node.js**（バージョン 18 以上）がシステムにインストールされている
- **React** の基本的な知識がある - React が初めての場合は [React ドキュメント](https://react.dev/learn) を参照してください
- テキストエディタまたは IDE（[Visual Studio Code](https://code.visualstudio.com/) を推奨）

## クイックスタート

最も速く始める方法はスキャフォールディングツールを使用することです:

```bash
npm create playcanvas@latest -- -t react-ts
```

これにより、すべてがセットアップされたすぐに使えるプロジェクトが作成されます。ローカルにインストールせずに試したい場合は、[オンラインプレイグラウンド](https://playcanvas-react.vercel.app/new) もご利用いただけます。

## 次のステップ

準備ができましたか？ まずは [インストールガイド](./installation) で開発環境をセットアップし、[シーンの構築](../building-a-scene) チュートリアルで最初の 3D アプリケーションを作成しましょう。

より高度なトピックや例については、[PlayCanvas React ドキュメント](https://playcanvas-react.vercel.app/docs) と [サンプルギャラリー](https://playcanvas-react.vercel.app/examples) をご覧ください。
