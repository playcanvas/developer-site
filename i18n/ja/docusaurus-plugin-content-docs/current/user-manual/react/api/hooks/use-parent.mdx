# `useParent`

`useParent` フックは React コンポーネント内から親の [Entity](https://api.playcanvas.com/engine/classes/Entity.html) にアクセスする手段を提供します。親 Entity と対話する必要があるコンポーネントに不可欠です。

## 基本的な使い方

```jsx copy
import { useParent } from '@playcanvas/react/hooks'

function MyComponent() {
  const parent = useParent()
  
  // 親 Entity のプロパティにアクセス
  console.log('Parent name:', parent.name)
  console.log('Parent position:', parent.position)
  
  return null
}
```

## 戻り値

```tsx asTypedoc
import { Entity } from 'playcanvas'
type $ = {
  /**
   * The PlayCanvas Entity instance
   */
  parent: Entity
}
export default $;
```

## 要件

このフックは `<Entity>` コンポーネント内で使用する必要があります。このコンテキスト外で使用するとエラーをスローします。

```jsx
// ✅ 正しい使い方
<Entity>
  <MyComponent /> {/* useParent はここで動作します */}
</Entity>

// ❌ 間違った使い方
<MyComponent /> {/* useParent はエラーをスローします */}
```

## 例

### Entity プロパティへのアクセス

```jsx copy
import { useParent } from '@playcanvas/react/hooks'

function EntityInfo() {
  const parent = useParent()
  
  return (
    <div>
      <p>Entity Name: {parent.name}</p>
      <p>Position: {parent.position.toString()}</p>
      <p>Rotation: {parent.rotation.toString()}</p>
      <p>Scale: {parent.scale.toString()}</p>
    </div>
  )
}
```
