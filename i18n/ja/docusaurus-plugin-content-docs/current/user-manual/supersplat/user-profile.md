SuperSplatの各クリエイターには`https://superspl.at/user?id=<username>`に公開プロファイルが用意されます。プロファイルには、アバター、表示名、自己紹介、追加されたソーシャルリンク、そして公開したすべての**Public**スプラットのグリッドが表示されます。

プロファイルは、すべての[シーンページ](/user-manual/supersplat/scene-page)と[Explore](/user-manual/supersplat/explore)グリッドのすべてのカードに表示される**著者チップ**からリンクされます — 著者名をクリックするとプロファイルへ移動します。

プロファイルの閲覧は匿名で可能で、PlayCanvasアカウントは不要です。

## 表示される内容

| セクション | 含まれるもの |
|---------|------------------|
| **アバターと表示名** | クリエイターのPlayCanvasアカウントのアバターと、選択した表示名。 |
| **自己紹介** | クリエイター自身が書いた短いプロフィール紹介。 |
| **ソーシャルリンク** | クリエイターが追加した外部プロフィール（X、LinkedIn、個人サイトなど）へのリンク。 |
| **スプラットグリッド** | クリエイターが公開したすべての**Public**スプラットを、[Explore](/user-manual/supersplat/explore)と同じカードUIで表示。任意のカードをクリックすると、その[シーンページ](/user-manual/supersplat/scene-page)が開きます。 |

Unlistedのスプラットは公開プロファイルには表示されません — クリエイター本人だけが[Manageページ](/user-manual/supersplat/manage)から確認できます。
