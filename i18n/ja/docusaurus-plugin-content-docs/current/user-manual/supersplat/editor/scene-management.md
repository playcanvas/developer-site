title: シーン管理
description: "SuperSplatで複数のスプラットシーンを選択、表示、ソロ表示、名前変更、削除、位置合わせ、結合する方法です。"

インポートした各ファイルは、**Scene Manager**に個別の行として表示されます。複数のキャプチャを組み合わせたり、異なるバージョンを比較したり、別々に撮影した領域を結合前に位置合わせしたりできます。

## アクティブシーン

行をクリックすると、そのシーンがアクティブになります。次の操作はアクティブシーンを対象にします。

- ガウシアンの選択と編集
- **Transform**、**Color**、**Splat Data**パネル
- カメラのフォーカスと測定ツール

一度にアクティブにできるシーンは1つだけです。行を選択しても表示状態は変わりません。

## シーンの追加

**File > Import**、Scene Managerヘッダーのインポートボタン、または対応ファイルのビューポートへのドラッグを使用します。インポートは現在のプロジェクトへコンテンツを追加し、読み込み済みのシーンを置き換えません。

**File > New**は動作が異なります。現在のプロジェクトにデータがある場合は確認した後、新しい空のプロジェクトを開始します。

対応フォーマットと複数ファイルのインポート要件については、[インポートとエクスポート](import-export.md)を参照してください。

## 表示とSoloモード

シーン行の目のアイコンをクリックすると、表示と非表示を切り替えられます。非表示のシーンはスプラットのエクスポートから除外されるため、シーンを結合する前に表示状態を確認してください。

表示設定を変更せずに1つのシーンだけを編集する場合は、Scene Managerヘッダーの**Solo**ボタンを使用します。

1. シーンを選択します。
2. **Solo**を有効にして、ほかのすべてのシーンを非表示にします。
3. 別の行を選択すると、表示するシーンが切り替わります。
4. **Solo**を無効にすると、有効化前の表示状態が復元されます。

Soloモード中にシーンをインポートした場合、その行を選択すると表示できます。

## シーン名の変更と削除

シーン名をダブルクリックすると名前を変更できます。名前の変更は元に戻すこともやり直すこともできます。

シーン行のごみ箱アイコンをクリックすると、シーン全体がプロジェクトから削除されます。シーンの削除は元に戻せません。編集履歴に記録されて元に戻せる、選択したガウシアンの削除とは異なります。後でシーンが必要になる可能性がある場合は、先に`.ssproj`のコピーを保存してください。

## 複数キャプチャの組み立て

一般的な複数シーンのワークフローは次のとおりです。

1. すべてのキャプチャをインポートします。
2. Soloモードを使用して、各キャプチャを個別に確認およびクリーンアップします。
3. [変形ツール](transforming-splats.md)を使用して、各シーンを移動、回転、拡大縮小します。
4. 複数のシーンを同時に表示し、いくつかのカメラアングルから位置合わせを確認します。
5. 含めないシーンを非表示にします。
6. **File > Export > PLY**を選択して、表示中のシーンを1つのPLYファイルに結合します。

**File > Save**では、結合済みのPLYではなく編集可能な`.ssproj`プロジェクトが保存されます。[プロジェクトの管理](managing-projects.md)も参照してください。
