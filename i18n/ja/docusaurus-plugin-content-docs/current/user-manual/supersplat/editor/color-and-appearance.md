---

**Color**パネルでは、アクティブスプラットの外観を非破壊で調整できます。右ツールバーのカラーアイコンから開き、Scene Managerで調整するスプラットを選択します。

色調整は、選択したガウシアンだけではなく、アクティブスプラット全体に適用されます。領域ごとに異なる調整を行う場合は、先に[選択範囲を新しいスプラットへ分離](editing-splats.md#duplicate-and-separate-a-selection)してください。

## コントロール

| コントロール | 効果 |
|---|---|
| **Tint** | スプラットの色に選択した色を乗算します。白では元の色が変わりません。 |
| **Temperature** | 結果を寒色または暖色方向へ移動します。 |
| **Saturation** | 色の強さを下げたり上げたりします。`1`では彩度が変わりません。 |
| **Brightness** | スプラットを暗くまたは明るくします。`0`では明るさが変わりません。 |
| **Black Point** | 黒としてマッピングされる最暗値を上げ、シャドウのクリッピングを強めます。 |
| **White Point** | 白としてマッピングされる最明値を下げ、ハイライトのクリッピングを強めます。 |
| **Transparency** | 元の値に対してスプラットの不透明度を下げたり上げたりします。 |

Black PointをWhite Pointより高く設定することはできません。Colorパネルのヘッダーにあるリセットボタンを使用すると、すべてのコントロールが初期値に戻ります。調整とリセットは元に戻すこともやり直すこともできます。

## 実用的な色補正ワークフロー

1. **Settings**でビューポートのトーンマッピングと背景を設定し、意図した表示条件でシーンを確認します。
2. **Temperature**で全体の寒暖を補正します。
3. **Brightness**を調整し、次に**Black Point**と**White Point**で階調範囲を設定します。
4. **Saturation**と**Tint**で最終的なカラーバランスを整えます。
5. 複数のカメラアングルから結果を確認します。球面調和関数の色は視線方向によって変化する場合があります。

ビューポートのトーンマッピング、露出、背景は表示設定です。Colorパネルはスプラット自体を変更します。

## 保存とエクスポート時の動作

`.ssproj`プロジェクトに保存すると、これらのコントロールは編集可能なスプラット設定として保持されます。スプラット形式へエクスポートすると、調整がエクスポート先の色と不透明度データに適用されます。

色などの属性をガウシアン単位で確認および選択するには、[Splat Dataパネル](data-panel.md)を使用します。スプラットの外観を整えた後でビューポートカメラをアニメーション化するには、[Timeline](timeline.md)を参照してください。
