SuperSplatでは、インポートしたスプラット全体またはその中で選択したガウシアンだけを変形できます。現在の選択状態によって対象が決まります。

| アクティブスプラットの状態 | 変形の対象 |
|---|---|
| ガウシアンが選択されていない | アクティブスプラット全体 |
| 1つ以上のガウシアンが選択されている | 選択したガウシアンのみ |

スプラット全体を移動する場合は、変形前に**Select > None**を選択してください。

## 変形ツール

アクティブスプラットを選択し、下部ツールバーからツールを選びます。

- **Move**（`1`）は対象を移動します。
- **Rotate**（`2`）は対象を回転します。
- **Scale**（`3`）は対象を拡大縮小します。

ビューポート内のギズモハンドルをドラッグします。正確な値を指定するには、Scene Managerの下にある**Transform**パネルで**Position**、**Rotation**、均一な**Scale**を編集します。変形操作は元に戻すこともやり直すこともできます。

ギズモは最初、スプラットのローカル座標空間で動作します。`Shift + C`（または下部ツールバーの座標空間トグル）で、ローカルとワールドの座標空間を切り替えられます。

## ピボットの配置

ピボットは変形の原点、つまり回転や拡大縮小の中心を示すと同時に、スプラットのローカル座標フレームを定義します。

- Move、Rotate、Scaleのいずれかがアクティブな間に表示サーフェスをダブルクリックすると、ピボットがその位置へ移動します。
- 下部ツールバーの**Reset Pivot**をクリックすると、ピボットがスプラットの原点に戻り、向きもリセットされます。`Shift`を押しながらクリックすると、代わりに境界の中心（スプラット全体の中心、ガウシアンが選択されている場合はその選択範囲の中心）にピボットが配置されます。
- [Orientツール](#orient-a-splat-to-the-grid)を使うと、スプラットのサーフェス上で指定した平面からピボットを設定できます。

ピボットを移動しただけではスプラットは移動せず、その後に変形コントロールを使用したときに作用します。ピボットの変更は元に戻すことができ、カスタムピボットは`.ssproj`ファイルにプロジェクトとともに保存されます。

## スプラットをグリッドに合わせる {#orient-a-splat-to-the-grid}

Orientツールは、サーフェス上で指定した平面をグリッドに合わせることで、回転がずれたスプラット（Z-upのインポート、傾いたキャプチャ、上下逆のスキャンなど）を水平にします。

1. 下部ツールバーで**Orient**を有効にします。
2. 床やテーブルの天面など、水平であるべきサーフェス上の3点をクリックします。
3. 配置した点をクリックし、移動ギズモで位置を微調整します。`Delete`または`Backspace`でアクティブな点を削除、**Clear**ですべての点を削除、`F`で点をビューポートにフレーミングできます。
4. **Align to Grid**を押します。指定した平面がグリッド平面にぴったり重なるように、アクティブスプラット全体が回転・移動します。指定した点を中心に回転するため、クリックした領域はその場に保たれます。

平面のうちカメラに向いている側が上になります。結果が上下逆になった場合は、平面の下側にカメラを回り込ませてもう一度**Align to Grid**を押してください。指定した平面がスプラット自身の軸から3度以内にある場合、回転はその軸にスナップするため、一般的なZ-upのインポートは正確な90度回転になります。

位置合わせは、ガウシアンの選択状態にかかわらずアクティブスプラット全体に適用され、1回の元に戻せる操作として記録されます。ツールがアクティブな間は、グリッドが非表示でも表示されます。対象となる平面は**Settings**の**Grid Plane**設定（デフォルトはXZ、ほかにXY/YZ）に従います。

3点を配置すると、下部ツールバーのピボットボタンが**Set Pivot**に変わります。このボタンはスプラットを動かさずに、指定した平面をスプラットのピボットとローカルフレームとして保存します。以降、ローカル空間のギズモと**Transform**パネルはこのフレームで動作します。スプラット自体を回転させずに、サーフェス独自の軸に沿って変形したい場合に便利です。

## 複数スプラットの位置合わせ

別々にキャプチャしたスプラットを位置合わせするには、次の手順を使用します。

1. 各キャプチャを個別のスプラットとしてインポートします。
2. 最初のスプラットを選択し、基準として固定したままにします。
3. 位置合わせするスプラットを選択し、**Select > None**を選んでスプラット全体を変形対象にします。
4. Move、Rotate、Scaleを使用し、複数のカメラアングルから結果を確認します。
5. 表示を切り替えるかSoloモードを使用して、スプラットを比較します。

グリッドの単位はメートルで、主目盛りは1 m、副目盛りは0.1 mです。数値によるサイズ確認が必要な場合は、**Settings**で**Show Bounding Box**と**Show Dimensions**を有効にします。

## スプラットの測定とリスケール

Measureツールは2つのサーフェス点間の距離を表示し、既知の実寸に合わせてアクティブスプラット全体をリスケールできます。

1. 下部ツールバーで**Measure**を有効にします。
2. サーフェスをクリックして最初のマーカーを配置し、もう一度クリックして2つ目を配置します。
3. いずれかのマーカーをクリックし、移動ギズモで位置を微調整します。
4. 下部コントロールの上にあるツールバーで測定距離を確認します。**Settings**で**Show Dimensions**を有効にすると、ビューポート内の線に沿って距離のラベルも表示されます。
5. スプラットのスケールを合わせるには、そのフィールドに実際の距離を入力します。2つのマーカーの中点を中心に、アクティブスプラット全体が均一にリスケールされます。

Measureがアクティブな間、`Delete`または`Backspace`でアクティブなマーカーを削除でき、`F`で配置したマーカーをフレーミングできます。`Escape`でツールを終了します。距離の変更は元に戻せます。

## 表示中のスプラットの結合

スプラットの位置合わせとクリーンアップが完了したら、含めるすべてのスプラットを表示し、**File > Export > PLY**を選択します。表示中のスプラットが1つのファイルに結合されます。

**File > Save**では、結合済みのPLYではなく、各スプラットを編集可能な状態で`.ssproj`プロジェクトに保存します。[スプラットの管理](scene-management.md)および[プロジェクトの管理](managing-projects.md)も参照してください。
