Windows, Mac OS X, Linuxなどの主要なデスクトップOSのそれぞれのネイティブ実行可能ファイルに、PlayCanvasのappを変換することが可能です。これを行うためには[NW.js](https://nwjs.io/)を使用することをお勧めします。実行ファイルを生成するための手順は[NW.js Wiki](https://github.com/nwjs/nw.js/wiki/How-to-package-and-distribute-your-apps)からご確認ください。
