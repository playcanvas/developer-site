**シーン**はPlayCanvasアプリケーションの基盤です。ユーザーが体験する3Dワールドを定義し、ゲームやインタラクティブ体験を構成するすべてのエンティティ、コンポーネント、設定が含まれます。

## シーンの構成要素

すべてのシーンには以下が含まれます。

- **エンティティ** - シーン内のオブジェクトで、階層構造で整理されています。エンティティはキャラクター、小道具、ライト、カメラ、その他あらゆる要素を表すことができます。
- **コンポーネント** - エンティティに動作と外観を与える構成要素です。コンポーネントはレンダリング、物理、オーディオ、ユーザーインターフェースなどを処理します。
- **シーン設定** - シーン全体に影響するグローバルプロパティで、物理の重力、アンビエントライティング、フォグ、スカイボックスなどが含まれます。
