---
title: テクスチャインスペクター
sidebar_position: 4
description: Assets Pane で選択したテクスチャ Asset の名前と設定を行う Texture Inspector です。
---

![Texture Inspector](/img/shader-editor/inspector-pane-texture.png)

テクスチャインスペクターは、テクスチャが設定される場所です。

テクスチャの詳細にアクセスするには、[アセットパネル](/shader-editor/window-layout/assets-pane)で選択してください。

- Name - テクスチャの名前を指定します。
