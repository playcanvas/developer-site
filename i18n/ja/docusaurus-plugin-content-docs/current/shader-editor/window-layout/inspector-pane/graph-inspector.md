---

![Graph Inspector](pathname:///img/shader-editor/inspector-pane-graph.png)

グラフインスペクターは、グラフの設定が構成される場所です。

グラフのインスペクタにアクセスするには、[アセットペイン](/shader-editor/window-layout/assets-pane)で選択します。

| 絞り込み制御         | 説明                                           |
| --------------- |---------------------------------------------- |
| Name            | グラフの名前                               |
| Label           | グラフの説明文                   |
| \+ NEW MATERIAL | 現在のグラフから新しいマテリアルを作成します。 |
