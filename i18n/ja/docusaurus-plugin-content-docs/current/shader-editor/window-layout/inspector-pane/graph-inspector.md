![グラフインスペクター][1]

グラフインスペクターは、グラフの設定が構成される場所です。

グラフのインスペクタにアクセスするには、[アセットペイン][2]で選択します。

| 項目         | 説明                                           |
| --------------- |---------------------------------------------- |
| Name            | グラフの名前                               |
| Label           | グラフの説明文                   |
| \+ NEW MATERIAL | 現在のグラフから新しいマテリアルを作成します。 |
