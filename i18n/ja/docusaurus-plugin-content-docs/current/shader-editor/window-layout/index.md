title: ウィンドウの配置
sidebar_position: 1.2
---

シェーダーエディターはいくつかのサブウィンドウまたはペインで構成されています。以下でペインについてハイライトし、それぞれの簡単な説明をします。

![window-layout](/img/shader-editor/window-layout.png)

| ペイン (Pane)                | 説明                                                       |
| ------------------- | ----------------------------------------------------------------- |
| [インフォペイン (Info Pane)](/shader-editor/window-layout/info-pane)      | マウスカーソルのあるUI要素の情報を表示します。      |
| [ノードペイン (Nodes Pane)](/shader-editor/window-layout/nodes-pane)     | グラフ作成時に使用可能な組み込みノードを一覧表示します。      |
| [インスペクターペイン (Inspector Pane)](/shader-editor/window-layout/inspector-pane) | 現在選択されているアイテムの詳細を表示します。                     |
| [アセットペイン (Assets Pane)](/shader-editor/window-layout/assets-pane)    | プロジェクトのグラフ、マテリアル、テクスチャが表示される場所です。    |
| [グラフエディター](/shader-editor/window-layout/graph-editor)   | 選択したグラフが表示され、ノードやエッジを編集できます。 |
| [プレビューペイン (Preview Pane)](/shader-editor/window-layout/preview-pane)   | 現在選択されているグラフやマテリアルのプレビューを表示します。   |
