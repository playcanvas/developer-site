---

グラフは、シェーダーエディターの中心的な概念です。ノードは処理を行い、エッジはノードを接続します。シェーダーエディターでは、グラフを利用してレンダリング用のシェーダプログラムを生成します。

シェーダーエディターには、[マテリアルグラフ](/shader-editor/overview/graphs/material-graphs)と[サブグラフ](/shader-editor/overview/graphs/sub-graphs)の2種類のグラフが存在します。

グラフの管理方法については、[グラフエディター](/shader-editor/window-layout/graph-editor)のドキュメントを参照してください。
