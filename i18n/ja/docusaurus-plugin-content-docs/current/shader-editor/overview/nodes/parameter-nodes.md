---
title: パラメーターノード
sidebar_position: 1
description: ランタイムで設定可能なマテリアル値のための Parameter ノードと、Shader Editor の Material Inspector での表示です。
---

パラメーターノードは、マテリアル内の値をランタイムで設定可能にする入力ノードです。グラフに追加されたパラメーターノードは、マテリアルインスペクターパネルに表示されます。

パラメータの編集の詳細については、[インスペクターパネル](/shader-editor/window-layout/inspector-pane)のドキュメントを参照してください。
